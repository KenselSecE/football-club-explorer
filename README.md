# Football Club Explorer (FCE)

Football Club Explorer is a Next.js web application that allows users to explore
football clubs from different leagues and countries, view club details, and save
their favorite teams.

## Features

- Browse football clubs with clean card-based UI
- View detailed information for each club
- Explore leagues and see how many clubs belong to each league
- Add and remove clubs from Favorites
- Responsive layout using CSS Modules
- Static data loading with JSON

## Technologies Used

- Next.js (App Router)
- React
- TypeScript
- CSS Modules
- JSON for mock data

## Project Structure

- `/app` – Pages and routes
- `/components` – Reusable UI components
- `/public/data` – Clubs data (JSON)
- `/types` – TypeScript interfaces
