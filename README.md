# Football Club Explorer (FCE)

Football Club Explorer (FCE) is a web application that allows users to explore football clubs from different leagues and countries around the world. The project demonstrates key frontend development concepts such as component-based UI design, state management, routing, and data handling using modern web technologies.

# Features
- Browse football clubs by league or country
- View basic club details (founded year, league, trophies)
- Local JSON used to simulate API data
- Clean UI with reusable components
- Fast navigation using Next.js routing

# Tech Stack
- Next.js
- React
- TypeScript
- CSS Modules
- Local JSON data
